The [ErgoDox EZ](https://ergodox-ez.com) is a commercially assembled version of the ErgoDox, available with a 2-year warranty and fully customizable layout.

## Configuring the EZ
The ErgoDox EZ can be configured using the **[Oryx Configurator](https://configure.zsa.io/)** without writing any code. It also supports flashing custom **QMK firmware** for power users.
